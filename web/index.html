Cloud FaaS
Containerized Functions

Run tiny cloud functions from your browser.

This mini FaaS accepts Python, JavaScript, C, C++, Java, and PHP code, executes each request in an isolated Docker container, captures the output, and removes the container after the run.

Docker-Isolated 128 MB Memory Cap 50% CPU Quota 10 Second Timeout

Run a function

Choose a language, then either paste code directly or upload a function file.

Paste plain source code here when you want to run the JSON endpoint. Uploaded or pasted code that waits for input will switch into the same live input flow.
Accepted extensions: .py, .js, .c, .cpp, .java, .php. Uploaded files that wait for input will switch into the same live input flow.
Upload an image or supporting asset that your function should read from the current working folder during the run.
Use inline mode for normal stdout. For files, either print a JSON object with faas_download.base64 or write a file into /function/faas_downloads. Direct download streams the generated artifact straight back into the browser.
